# GoLazy

GoLazy is a convention-first web framework for Go.

This repository currently contains the application, controller, dispatch, and
routing primitives extracted from the sample application:

```text
golazy.dev/lazyapp
golazy.dev/lazycontroller
golazy.dev/lazydispatch
golazy.dev/lazyroutes
golazy.dev/lazysupport/inflection
```

Its canonical Go module path is:

```text
golazy.dev
```

The `golazy.dev` vanity import is served by the website repository and resolves
to `https://github.com/golazy/golazy`.
